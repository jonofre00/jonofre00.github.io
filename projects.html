Projects

Here you will find information on projects I have worked on.

Parking Assignment API

I created an API that allowed a customer to submit bulk requests for parking permits. The API would accept a listing of permit requests and would return the assigned parking number in the response. In the backend, this also automatically filled our fulfillment tables necessary to ship. By doing this project, it removed the need to communicate via email and the manual entry and assignment for each permit in the email.

Storefront Integration

I created a webhook that would automatically create an order into our ERP system whenever an order was submitted into the company system. This created the order and the shipping record. The result of this project raised productivity significantly by eliminating the need to manually enter each submitted order into the ERP system.
